# Prosthetichand
Patient-specific EMG hand gesture recognition (ANN) for Tripod/Pinch/Power Grip. Bioamp EXG Pill data denoised via VMD (high-SNR IMFs) removing powerline noise/baseline wander. Time/freq features (mean,std,RMS,ZC,PSD,energy) beat CWT. FNN: 84.38% 10-fold CV. Hardware prototype + AWS cloud enables PC-free real-time prosthetics.
